 Free Code Camp Front-End Project
 
jQuery is used to get the top ten articles of any Wikipedia search and display them to the user. You may also click the “Random Article” button to get a random Wikipedia article. A greedy regular expression (^.*[.?!]\s) extracts the first sentence from all articles in order to provide a summary for each article.
